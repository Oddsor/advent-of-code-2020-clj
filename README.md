## Building the notebooks

This command will build notebooks as html into the `public`-folder
```
clojure -Xbuildnotebooks
```

## Outdated dependencies:
To see outdated dependencies:
```
clojure -M:outdated
```
To upgrade:
```
clojure -M:update
```
